# Source format

A source is something external you read: a paper, a dataset. The file itself lives in `sources/`; the summary lives in the wiki.

File: `wiki/<short-name>.md`, type `source`.

**Record:**

- what it is, and where the file sits (`sources/...`)
- the points that matter to this project, in your own words. Paraphrase; do not paste long passages.
- what it bears on: the parameters, findings, or decisions it touches, by link
- any conflict with what the project already holds, stated plainly

A source summary is wiki: update it as your understanding improves. The source file itself does not change.
