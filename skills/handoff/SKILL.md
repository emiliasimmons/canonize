---

Write a short handoff note so another session, or a later you, can pick up the thread. Save it to a temp folder; it is scratch, not part of the project, and it is fine for it to disappear once it has done its job.

Whatever already lives in the project (sources, findings, MDRs, provenance, wiki pages) do not recopy it; point to it by path. The note carries only what has not been written down yet: where things stand, what is still open, and anything you settled but did not save.

Name the skills the next session should reach for. Leave out anything sensitive.

If you are given a focus, an idea to go chase or findings to bring back, shape the note around it.
